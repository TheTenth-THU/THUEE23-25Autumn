## Semiconductor Memory <br>半导体存储器

### Volatile memory <br>易失存储器

#### SRAM

**SRAM (static random-access memory, 静态随机存取存储器)** 是一种使用**触发器 (flip-flop)** 作为基本存储单元的半导体存储器，其存储单元基本结构即两个反相器交叉连接形成的双稳态电路。

```tx
| SRAM ||
| :----------- | :---- |
| **刷新需求** | 在电源电压维持期间，SRAM 存储单元可以自行保持数据，|\
|              | **不需要周期性刷新** |
| **写入速度** | 写入仅与晶体管建立时间相关，写入速度高 |
| **读取速度** | 使用双位线直接访问，读取速度高 |
| **单元面积** | 每个存储单元需 **6 个晶体管**，占用面积较大 |
```

由于 SRAM 具有高速读写和无需刷新等优点，常用于处理器的**一级缓存 (L1 cache)** 和**二级缓存 (L2 cache)** 中，以提高数据访问速度。

#### DRAM

**DRAM (dynamic random-access memory, 动态随机存取存储器)** 是一种使用**电容 (capacitor)** 作为基本存储单元的半导体存储器，其存储单元基本结构为一个电容和一个访问晶体管。

```tx
| DRAM ||
| :----------- | :---- |
| **刷新需求** | 由于电容会随时间泄漏电荷，**需要周期性刷新**以维持数据完整性 |
| **写入速度** | 写入需充电至电容，写入速度较慢 |
| **读取速度** | 单管 DRAM 单元的**读出是破坏性的**，读出时需要进行刷新，读取速度较慢 |
| **单元面积** | 每个存储单元仅需 **1 个晶体管和 1 个电容**，占用面积较小 |
```

由于 DRAM 具有高密度和低成本等优点，常用于计算机的**主存 (main memory)** 中，以提供大容量的数据存储。

> [!note] eDRAM
> **eDRAM (embedded DRAM, 嵌入式动态随机存取存储器)** 是一种集成在处理器芯片上的 DRAM，结合了 DRAM 的高密度和 SRAM 的高速访问特点。
> 
> 将 DRAM 集成在片上需要解决**电容制造工艺**和**刷新管理**等挑战。eDRAM 通过优化电容设计和刷新机制，实现了**较高的存储密度和较低的功耗**，与 SRAM 相比可节省 3~4 倍的面积，抵消了工艺成本。
> 
> eDRAM 通常用于处理器的**三级缓存 (L3 cache)** 甚至四级缓存 (L4 cache)，以提高缓存容量和性能。

### Storage memory <br>非易失存储器

非易失存储器通常用于**长期存储数据**，断电后数据仍然保持不变。常见的非易失存储器包括：
+ **FeRAM (ferroelectric RAM, 铁电随机存取存储器)**：利用铁电材料的极化特性存储数据。
+ **MRAM (magnetoresistive RAM, 磁阻随机存取存储器)**：利用磁性材料的磁阻效应存储数据。
+ **PCM (phase-change memory, 相变存储器)**：利用材料的相变特性存储数据。
+ **Flash Memory (闪存)**：利用浮栅晶体管存储数据，广泛应用于固态硬盘 (SSD) 和 USB 闪存驱动器。

## Memory Hierarchy <br>存储器的层次化

### Temporal & Spatial locality <br>时间局部性与空间局部性

程序在访问数据时通常表现出**时间局部性 (temporal locality)** 和**空间局部性 (spatial locality)**：
+ **时间局部性**指程序倾向于在短时间内多次访问相同的数据或指令，**最近**被访问的数据很可能**在不久的将来再次被访问**，如运算常量、循环变量、卷积模板等；
+ **空间局部性**指程序倾向于访问**相邻地址**的数据或指令，访问某个数据后，很可能会**访问其附近的数据**，如数组元素、结构体成员等。
