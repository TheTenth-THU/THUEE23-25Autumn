## 一维线性谐振子运动方程

### 一维线性谐振子运动方程的建立

经典力学下，受到回复力 $F = -kx$  的谐振子势能为 $V(x) = \dfrac{1}{2} kx^{2}$，产生谐振的角频率为 $\omega = \sqrt{\dfrac{k}{m}}$；类似地，给定势场 $V(x) = \dfrac{1}{2} \mu \omega^{2} x^{2}$，考察其中的波函数。

[[Schrödinger 方程#^DingtaiSchrodingerFangcheng|定态 Schrödinger 方程]]为
$$
\dfrac{\dif^{2} \psi(x)}{\dif x^{2}} + \dfrac{2\mu}{\hbar^{2}} \left( E - \dfrac{1}{2} \mu \omega^{2} x^{2} \right) \psi(x) = 0
$$
定义**无量纲变量 $\xi = \sqrt{\dfrac{\mu \omega}{\hbar}} x$，$\lambda = \dfrac{2E}{\hbar \omega}$**，则方程化为
$$
\mark{ \dfrac{\dif^{2} \psi(\xi)}{\dif \xi^{2}} + \left( \lambda - \xi^{2} \right) \psi(\xi) = 0 }
$$

> [!note] 线性谐振子方程解的试探
> 线性谐振子方程中，$\lambda$ 是与能量 $E$ 相关的参量。下面的例子显示，$\lambda(E)$ 只能取一系列特殊的**分立值**，才能保证方程具有**有限**的解。

### 无量纲一维线性谐振子运动方程的求解

#### 方程解的渐近行为分析

观察到 $\xi \to \pm \infty$ 时，方程近似为 $\dfrac{\dif^{2} \psi(\xi)}{\dif \xi^{2}} - \xi^{2} \psi(\xi) = 0$，其通解为 
$$
\psi(\xi) = A \e^{-\xi^{2}/2} + B \e^{\xi^{2}/2}
$$
由于 $\e^{\xi^{2}/2}$ 在无穷远处发散，因此**有限性**要求 $B = 0$，即取渐近解 $\psi(\xi) \sim \e^{-\xi^{2}/2}$。

设方程的一般解为 $\psi(\xi) = \e^{-\xi^{2}/2} H(\xi)$，回代得到
$$
\dfrac{\dif^{2}}{\dif \xi^{2}} H(\xi) - 2 \xi \dfrac{\dif}{\dif \xi} H(\xi) + (\lambda - 1) H(\xi) = 0
$$
这是 **Hermite 方程**，其解 $H(\xi)$ 称为 **Hermite 多项式**。设 $H(\xi)$ 可展为幂级数 $\sum\limits_{\nu=0}^{\infty} a_{\nu} \xi^{\nu}$，代入得到
$$
\sum\limits_{\nu=0}^{\infty} (\nu + 2) (\nu + 1) a_{\nu+2} \xi^{\nu} - 2 \sum\limits_{\nu=0}^{\infty} \nu a_{\nu} \xi^{\nu} + (\lambda - 1) \sum\limits_{\nu=0}^{\infty} a_{\nu} \xi^{\nu} = 0
$$
比较各幂次项系数，得到**递推关系**
$$
a_{\nu+2} = \dfrac{2\nu - \lambda + 1}{(\nu + 1)(\nu + 2)} a_{\nu},\quad \nu = 0, 1, 2, \cdots
$$

$\xi \to \infty$ 时，$H$ 一般**发散**，且由于 $\lim\limits_{ \nu \to \infty } \dfrac{a_{\nu+2}}{a_{\nu}} = \dfrac{2}{\nu}$，因此 $H(\xi)$ 的**渐近行为为 $\e^{\xi^{2}}$**，从而 $\psi(\xi)$ 的渐近行为为 $\e^{\xi^{2}/2}$，与有限性要求矛盾。无穷级数解总是发散，因此必须考虑退化为**有限级数**，即在某一项 $v = n$ 处，递推关系中分子为零，即 $2n - \lambda + 1 = 0$，从而有**量子化条件**
$$
\lambda = 2n + 1, \quad n = 0, 1, 2, \cdots
$$

#### Hermite 多项式与谐振子能级

根据[[Schrödinger 方程#^YuchengDingli|宇称定理]]，势场 $V(x) = \dfrac{1}{2} \mu \omega^{2} x^{2}$ 关于原点对称，则 $\psi(\xi)$ 应**有确定的宇称**，因此 $H(\xi)$ 也应有确定的宇称。由递推关系可知，$a_{0}$ 和 $a_{1}$ 可任意取值，因而
+ $a_{0}$ 不为零时，$a_{1} = 0$，表达式为
$$
H_{n}(\xi) = a_{0} + a_{2}\xi^{2} + a_{4}\xi^{4} + \cdots + a_{n} \xi^{n}
$$
此时 $n$ 为偶数，$\psi_{n}(\xi)$ 呈**偶宇称**；
+ $a_{1}$ 不为零时，$a_{0} = 0$，表达式为
$$
H_{n}(\xi) = a_{1}\xi + a_{3}\xi^{3} + a_{5}\xi^{5} + \cdots + a_{n} \xi^{n}
$$
此时 $n$ 为奇数，$\psi_{n}(\xi)$ 呈**奇宇称**。

适当选取最低阶系数，得到 **Hermite 多项式**表示为
$$
\mark{ H_{n} (\xi) = (-1)^{n} \e^{\xi^{2}} \dfrac{\dif^{n}}{\dif \xi^{n}} \e^{-\xi^{2}}, } \quad n = 0, 1, 2, \cdots
$$
而波函数 $\psi_{n}(\xi) = \e^{-\xi^{2}/2} H_{n}(\xi)$，**归一化**后得到**一维线性谐振子中的能量本征函数**
$$
\mark{ \psi_{n}(x) = N_{n} H_{n}(\alpha x) \e^{-\alpha^{2}x^{2}/2}, } \quad n = 0, 1, 2, \cdots
$$
其中 $N_{n} = \left( \dfrac{\alpha}{\sqrt{\pi} 2^{n} n!} \right)^{1/2}$，$\alpha = \sqrt{\dfrac{\mu \omega}{\hbar}}$；对应的**能量本征值**为
$$
\mark{ E_{n} = \left( n + \dfrac{1}{2} \right) \hbar \omega, } \quad n = 0, 1, 2, \cdots
$$

#### 谐振子波函数举例

常用的一维谐振子波函数有：
+ $n = 0$，称**基态**，能量 $E_{0} = \cfrac{1}{2} \hbar \omega$，波函数为
$$
\psi_{0}(x) = \left( \dfrac{\alpha}{\sqrt{ \pi }} \right)^{1/2} \e^{-\alpha^{2} x^{2}/2}
$$
^YiweiXiezhenziJitai
+ $n = 1$，称**第一激发态**，能量 $E_{1} = \dfrac{3}{2} \hbar \omega$，波函数为
$$
\psi_{1}(x) = \left( \dfrac{2\alpha}{\sqrt{\pi}} \right)^{1/2} \alpha x \e^{-\alpha^{2} x^{2}/2}
$$
+ $n = 2$，称**第二激发态**，能量 $E_{2} = \dfrac{5}{2} \hbar \omega$，波函数为
$$
\psi_{2}(x) = \left( \dfrac{\alpha}{2\sqrt{ \pi }} \right)^{1/2} (2\alpha^{2} x^{2} - 1) \e^{-\alpha^{2} x^{2}/2}
$$

> [!note] 谐振子相干态
> 将上述一维谐振子基态波包 $\psi_{0}(x) = \dfrac{\alpha^{1/2}}{\pi^{1/4}} \e^{-\alpha^{2}x^{2}/2}$ 平移到 $x = x_{0}$ 处，得到
> $$
> \psi(x) = \dfrac{\alpha^{1/2}}{\pi^{1/4}} \e^{-\alpha^{2} (x - x_{0})^{2}/2}
> $$
> 此即**相干态波包**初始时刻状态。可以由含时薛定谔方程证明，这一波包运动中形状保持不变，只是**波包中心周期性振荡**，与经典粒子的运动很相似，如
> $$
> |\varPsi(x, t)|^{2} = | \psi_{0} (x - x_{0} \cos\omega t) |^{2}
> $$
> 这表明谐振子相干态在量子力学中表现出类似经典谐振子的行为，因此在量子光学和量子信息领域有重要应用。

## 谐振子升降算符方法

### 升降算符与占有数表象

在一维谐振子问题中，能量算符可表示为
$$
\hat{H} = \dfrac{\hat{p}^{2}}{2\mu} + \dfrac{1}{2} \mu \omega^{2} \hat{x}^{2} = \hbar\omega \dfrac{1}{2} \bigg( \dfrac{\hat{p}^{2}}{\mu\hbar\omega} + \dfrac{\mu\omega \hat{x}^{2}}{\hbar} \bigg)
$$
其中 $\hbar\omega$ 的量纲为能量，因此可推知括号内两项亦无量纲，可定义**无量纲位置算符**与**无量纲动量算符**
$$
\hat{x}' = \sqrt{\dfrac{\mu\omega}{\hbar}} \hat{x} = \alpha \hat{x}, \qquad 
\hat{p}' = \dfrac{1}{\sqrt{\mu\hbar\omega}} \hat{p} = \dfrac{1}{\hbar\alpha} \hat{p}
$$
这样，能量算符化为
$$
\hat{H} = \hbar\omega \dfrac{1}{2} \left( \hat{p}'^{2} + \hat{x}'^{2} \right)
= \hbar\omega \dfrac{1}{2} \left( (\hat{x}' - \I \hat{p}')(\hat{x}' + \I \hat{p}') + 1 \right)
$$
此处括号中最后的 1 来自二者的对易关系 $\left[\hat{x}', \hat{p}'\right] = \I$。

> [!def.] 升降算符
> 定义**升算符 (creation operator)** $\hat{a}^{\dagger}$ 与**降算符 (annihilation operator)** $\hat{a}$ 为
> $$
> \hat{a} = \dfrac{1}{\sqrt{2}} \left( \hat{x}' + \I \hat{p}' \right), \qquad
> \hat{a}^{\dagger} = \dfrac{1}{\sqrt{2}} \left( \hat{x}' - \I \hat{p}' \right)
> $$
^ShengSuanfu-JiangSuanfu

记 $\hat{N} = \hat{a}^{\dagger}\hat{a}$，则能量算符可**用升降算符表示**为
$$
\hat{H} = \hbar\omega \left( \hat{a}^{\dagger} \hat{a} + \dfrac{1}{2} \right) = \hbar\omega \left( \hat{N} + \dfrac{1}{2} \right)
$$
显然，任一 **$\hat{H}$ 的本征态 $\ket{ \psi_{n} }$ 也是 $\hat{N}$ 的本征态**，其对应 $\hat{H}$ 的本征值为
$$
E_{n} = \hbar\omega \left( n + \dfrac{1}{2} \right)
$$
即知其对应 $\hat{N}$ 的本征值为 $n$，将该态简记为 $\ket{n}$。

谐振子的能量只能以 $\hbar\omega$ 为单位改变，可视为**每次吸收或释放一个能量量子 $\hbar\omega$**，因此 $n$ 被称为**占有数 (occupation number)**，表示谐振子中激发态的量子数。对应地，$\hat{N}$ 表象被称为**占有数表象 (number representation)**。

### 升降算符的作用

已知 $n = 0$ 对应**基态**能量 $E_{0} = \dfrac{1}{2} \hbar \omega$，此时
$$
\bra{0} \hat{a}^{\dagger} \hat{a} \ket{0} = \bra{0} \hat{N} \ket{0} = 0 \braket{ 0 | 0 } = 0
\quad\Longrightarrow\quad
\hat{a} \ket{0} = 0
$$
即**降算符 $\hat{a}$ 作用在基态 $\ket{0}$ 上得到零矢量**。

> [!example] 由降算符基态方程求解基态波函数
> 已知 $\hat{a} \ket{0} = 0$，将其写成位置表象，有
> $$
> \hat{a} \braket{ x | 0 }  = \dfrac{1}{\sqrt{2}} \left( \alpha \hat{x} + \dfrac{\I}{\hbar\alpha} \cdot(-\I\hbar) \dfrac{\dif}{\dif x} \right) \psi_{0}(x) = 0
> $$
> 即
> $$
> \dfrac{\dif}{\dif x} \psi_{0}(x) = - \alpha^{2} x \psi_{0}(x)
> $$
> 这是一个一阶常微分方程，可通过分离变量构造 $\cfrac{\dif\psi_{0}}{\psi_{0}}$ 积分求解，归一化解为
> $$
> \psi_{0}(x) = \left( \dfrac{\alpha}{\sqrt{ \pi }} \right)^{1/2} \e^{-\alpha^{2} x^{2}/2}
> $$

考察降算符 $\hat{a}$ 作用在任一能量本征态 $\ket{n}$ 上的结果，有
$$
\begin{align}
\hat{N} (\hat{a} \ket{n} ) &= (\hat{a}^{\dagger} \hat{a}) \hat{a} \ket{n} = (\hat{a} \hat{a}^{\dagger} -1) \hat{a} \ket{n}  = \hat{a} (\hat{a}^{\dagger} \hat{a} \ket{n}) - \hat{a} \ket{n} \\
&= \hat{a} n \ket{n} - \hat{a} \ket{n} = (n - 1) (\hat{a} \ket{n})
\end{align}
$$
因此，**$\hat{a} \ket{n}$ 也是 $\hat{N}$ 的本征态**，对应本征值为 $n - 1$；又由于**一维束缚态非简并**，故
$$
\hat{a} \ket{n} = c_{n} \ket{n - 1}
$$
假设 $\ket{n}$ 和 $\ket{n-1}$ 均已归一化，则有
$$
1 = \braket{ n-1 | n-1 } = \dfrac{1}{|c_{n}|^{2}} \bra{ n } \hat{a}^{\dagger} \hat{a} \ket{ n } = \dfrac{1}{|c_{n}|^{2}} \bra{ n } \hat{N} \ket{ n } = \dfrac{n}{|c_{n}|^{2}} 
$$
从而 $|c_{n}| = \sqrt{n}$，可取 $c_{n} = \sqrt{n}$，即
$$
\mark{ \hat{a} \ket{n} = \sqrt{n} \ket{n - 1}, }\quad n = 1, 2, 3, \cdots
$$

类似地，考察升算符 $\hat{a}^{\dagger}$ 作用在任一能量本征态 $\ket{n}$ 上的结果。直接由上面结论等式两侧同左乘 $\hat{a}^{\dagger}$，得到
$$
\hat{a}^{\dagger} \hat{a} \ket{n} = n \ket{n}  = \sqrt{n} \hat{a}^{\dagger} \ket{n - 1}
$$
即知
$$
\mark{ \hat{a}^{\dagger} \ket{n} = \sqrt{ n+1 } \ket{ n+1 },  }\quad n = 0, 1, 2, \cdots
$$

可见，**降算符 $\hat{a}$ 使谐振子态降低一个能级**，**升算符 $\hat{a}^{\dagger}$ 使谐振子态提升一个能级**，通过升降算符的反复作用，可由基态 $\ket{0}$ 生成任意激发态 $\ket{n}$，即
$$
\ket{n} = \dfrac{ ( \hat{a}^{\dagger} )^{n} }{ \sqrt{ n! } } \ket{0}, \quad n = 0, 1, 2, \cdots
$$

### 坐标、动量算符的占有数表象

由[[#^ShengSuanfu-JiangSuanfu|升降算符定义式]]，可解出无量纲位置算符与动量算符为
$$
\hat{x}' = \dfrac{1}{\sqrt{2}} ( \hat{a} + \hat{a}^{\dagger} ), \qquad
\hat{p}' = \dfrac{\I}{\sqrt{2}} ( \hat{a}^{\dagger} - \hat{a} )
$$
即**位置算符**与**动量算符**可表示为
$$
\hat{x} = \sqrt{ \dfrac{\hbar}{2\mu\omega} } ( \hat{a} + \hat{a}^{\dagger} ), \qquad
\hat{p} = \I \sqrt{ \dfrac{\mu\hbar\omega}{2} } ( \hat{a}^{\dagger} - \hat{a} )
$$

在**占有数表象**下，**位置算符的矩阵元**为
$$
\begin{align}
x_{m,n} = \bra{m} \hat{x} \ket{n} &= \sqrt{ \dfrac{\hbar}{2\mu\omega} } \bra{m} ( \hat{a} + \hat{a}^{\dagger} ) \ket{n}
= \sqrt{ \dfrac{\hbar}{2\mu\omega} } \left( \bra{m} \hat{a} \ket{n} + \bra{m} \hat{a}^{\dagger} \ket{n} \right)  \\
&= \sqrt{ \dfrac{\hbar}{2\mu\omega} } \left( \bra{m} \sqrt{n} \ket{n-1} + \bra{m} \sqrt{n+1} \ket{n+1} \right) \\
&= \sqrt{ \dfrac{\hbar}{2\mu\omega} } \left( \sqrt{n} \delta_{m, n-1} + \sqrt{n+1} \delta_{m, n+1} \right)
\end{align}
$$
类似地，**动量算符的矩阵元**为
$$
\begin{align}
p_{m,n} = \bra{m} \hat{p} \ket{n} &= \I \sqrt{ \dfrac{\mu\hbar\omega}{2} } \bra{m} ( \hat{a}^{\dagger} - \hat{a} ) \ket{n}
= \I \sqrt{ \dfrac{\mu\hbar\omega}{2} } \left( \bra{m} \hat{a}^{\dagger} \ket{n} - \bra{m} \hat{a} \ket{n} \right)  \\
&= \I \sqrt{ \dfrac{\mu\hbar\omega}{2} } \left( \bra{m} \sqrt{n+1} \ket{n+1} - \bra{m} \sqrt{n} \ket{n-1} \right) \\
&= \I \sqrt{ \dfrac{\mu\hbar\omega}{2} } \left( \sqrt{n+1} \delta_{m, n+1} - \sqrt{n} \delta_{m, n-1} \right)
\end{align}
$$
